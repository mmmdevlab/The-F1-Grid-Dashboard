## User Stories

### Overview Page

- **As a fan**, I want to see a countdown timer to the next race on the homepage, so that I know exactly how long until the next race weekend.
- **As a fan**, I want to see weather conditions for the next race location on the homepage, so that I can get a feel for race day conditions.
- **As a fan**, I want to see my watchlisted races featured on the homepage, so that my planned races are front and centre when I open the app.
- **As a fan**, I want to see my favourite drivers' points and championship position on the homepage, so that I can check in on them at a glance.
- **As a fan**, I want to see the top 5 driver championship standings on the homepage, so that I can see who's leading without having to search for it.

### Drivers Page

- **As a fan**, I want to browse all current drivers with their details (full name, nationality, number, and stats), so that I can explore the full grid.
- **As a fan**, I want to add or remove a driver from my favourites list, so that I can follow my favourites more quickly.
- **As a fan**, I want to see my favourite drivers pinned to the top of the page, so that I can access them without scrolling through the full list.

### Race Schedule Page

- **As a fan**, I want to see the full race calendar as cards showing the country, circuit name, and date, so that I know when and where all races are.
- **As a fan**, I want to add and remove races from my watchlist, so that I can keep track of races I plan to watch.
- **As a fan**, I want to see a countdown timer for each of my watchlisted races, so that I always know how much time is left.
- **As a fan**, I want past races to be visually distinguished from upcoming ones, so that I can quickly tell what's already happened.

### Dream Team Builder Page

- **As a fan**, I want to build a dream team by selecting a primary driver, secondary driver, favourite team, and favourite track, so that I can create my ideal F1 lineup.
- **As a fan**, I want to see a live preview of my selections as I pick them, so that I can visualise my team before saving.
- **As a fan**, I want to save and delete my dream team logs, so that I can keep a record and start fresh when my preferences change.

### Stretch Goals (Future)

- **As a fan**, I want to click on a driver card to see their full profile page, so that I can learn more about them.
- **As a fan**, I want to see constructor standings, so that I can track how my favourite team is performing.
- **As a fan**, I want to filter or search the driver list by name or team, so that I can find a specific driver quickly.
- **As a fan**, I want to see past race results on completed race cards, so that I can check how each round finished.
